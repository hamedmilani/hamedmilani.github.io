Helsa Nurse Scheduling Application

Nurse scheduling (or shift management) is a long-standing challenge in many hospitals and healthcare systems. While the problem has been widely studied in academia, especially in operations research, there are still very few software tools that are actually used by hospital staff.

Based on our interviews with nurses from three different cities in Iran, we found that nurse scheduling is still often done manually by one of the volunteer nurses. This process can be time-consuming and inefficient, especially when considering multiple constraints such as department-specific shift requirements, and fairness (based on seniority, weekend and night shifts, and individual preferences). This problem is a variation of the classic "assignment problem" in operations research. To address it, we develop an integer programming model in Python using the Gurobi optimization library. More technical details are available in the application catalog.

To generate the optimal nurse schedule, you need to define the following inputs: a) Nurse ID/Name/Seniority. b) Preferences for each shift (if available). c) Shift requirements (number of nurses needed per shift across the planning horizon). You can enter the inputs in one of two ways: 1) Manual entry: Add each input one by one inside the app. 2) File upload: Upload all inputs using three CSV files. You can also combine both methods, manual entries will override any matching entries from the uploaded files.

Download Templates

Download the following three input templates. Fill them in based on your own nurse list and needs, keep the header rows and the file names unchanged, and upload them before clicking Generate Optimal Schedule.

Currently, the app supports scheduling for up to 20 nurses.

Shift IDs: Morning: 1, Evening: 2, Night: 3.

Preference Levels: 0 means not available. 1 to 10 means level of preference (1 = least preferred, 10 = most preferred). No entry is treated as neutral (no preference).

File Upload

Upload CSV files for nurses, preferences, and requirements. Leave blank if using manual input.

Manual Input

Enter data for nurses, shift requirements, and preferences. Add multiple entries by clicking the "Add" buttons.

Nurses Information

Enter nurse details (ID, Name, Type - 0 for normal, 1 for senior).

Shift Requirements

Specify start date, end date, shift (1: Morning, 2: Evening, 3: Night), nurse type (0 for all, 1 for senior), minimum and maximum required nurses.

Preferences

Enter nurse ID, shift (1/2/3), date, and preference level (0–10). A preference level of 0 means unavailable; 1–10 indicates preference (10 is most preferred).
